Welcome to Space Shooter's Project Page

This is the project page for Space Shooter, my month 3 1GAM project. The Name Space Shooter is possibly a little misleading. The Game I have planned is a Dungeon Crawler type game, but set on a Space Ship. I plan to have a top down view, with LOTS of enemies and lots of projectiles. I would like to create a challenging game.

Plans

I plan to implement, several different enemy types. I plan to add level randomisation to give some replay-ability.
